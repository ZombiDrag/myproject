# Website Monitor

Простой мониторинг доступности сайтов: проверяет список сайтов, определяет их статус и записывает результат в лог с меткой времени. Запускается автоматически по расписанию через cron, упакован в Docker.

## Features

- Проверяет сразу несколько сайтов из списка (по одному на строку)
- Не падает, если сайт недоступен — ловит ошибку и продолжает проверять остальные
- Список сайтов вынесен в отдельный файл `sites.txt` — код менять не нужно
- Запускается автоматически по расписанию через cron
- Пишет результат в `log.txt` с меткой времени
- Упакован в Docker — работает одинаково на любой машине

## Tech Stack

- Python 3.12
- requests — HTTP-запросы к сайтам
- datetime — метки времени в логах
- Docker — упаковка в контейнер
- cron — запуск по расписанию

## Getting Started

1. Клонировать проект:
```bash
   git clone https://github.com/ZombiDrag/myproject.git
   cd myproject
```

2. Собрать Docker-образ:
```bash
   docker build -t checker-app .
```

3. Настроить автозапуск через cron (`crontab -e`):
```bash
   * * * * * cd /home/to/myproject && docker run -v $(pwd):/app checker-app
```
   Проверка запускается каждую минуту, логи пишутся в `log.txt`.
